## 🔥 How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/guptamanya18/protfolio.git

# 2. Go into the project directory
cd protfolio

# 3. Install dependencies
npm install

# 4. Run the development server
npm start
```

The app will run at `http://localhost:3000`.
